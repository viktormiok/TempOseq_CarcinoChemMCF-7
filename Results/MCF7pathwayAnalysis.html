Gene set enrichment analysis

The lists of the differentially expressed genes, for various conditions, are usually not the end point of the analysis. The second part of the analysis focus on the interpretation, where we are looking for the patterns among the differentially expressed genes. A lists of differentially expressed genes are easier to interpret if the genes exhibit similarity in their functionality. In this step analysis shifts from the level of single genes to sets of related genes, that are called pathways. Pathways are group of genes, which work together in order to process the signal and perform the certain function. Pathways are usually defined from libraries such as Gene Ontology (Ashburner et al., 2000) or KEGG (Ogata et al., 1999), which are based on previously accumulated knowledge. Definition of the pathways are always given a priori and are constructed without reference to the data.

Now, we are interested to test whether known biological process defined within the pathway are enriched (over-represented) in the selection of the genes from our experiment, which are considered to be differentially expressed. The function generates table which indicate for all the pathways from certain repository following characteristics: pathway id, pathway name, number of the genes in the pathway, number of the genes from our experiment in the pathway, number of significant genes from our selection, p-value and adjusted p-value. Significance analysis test is performed using cross table and employing fisher exact test. Cross table compare number of non and significant genes versus number of genes in and outside the pathway. Finally, due to the high-dimensionality of the data multiplicity correction is applied on the p-values across the genes.

# Function which generate pathway table
pathwayTable <- function(db, allG, sigG, noSigG){
  fin <- data.frame(matrix(ncol = 6, nrow = length(db)))
  for(i in 1:length(db)){
    fin[i,1] <- sub("_.*", "", names(db[i]))
    fin[i,2] <- gsub("_", " ", sub("^[^_]*", "", names(db[i])))
    fin[i,3] <- length(db[[i]])
    fin[i,4] <- length(intersect(as.numeric(db[[i]]),allG))
    fin[i,5] <- length(intersect(as.numeric(db[[i]]),sigG))
    CrossTab <- matrix(c(length(intersect(sigG,as.numeric(db[[i]]))), 
                         length(intersect(noSigG,as.numeric(db[[i]]))),
                         length(setdiff(sigG,as.numeric(db[[i]]))),
                         length(setdiff(noSigG,as.numeric(db[[i]])))), nrow = 2,)
    fin[i,6] <- fisher.test(CrossTab, alternative = "greater")$p.value
  }
  fin$X7 <- p.adjust(fin[,6], method ="fdr")
  colnames(fin) <- c("PathwayID", "PathName", "PathSize", "OurGen",
                     "NrSigGen", "Pval","adjPval")

  return(fin[order(fin[,7], fin[,6], decreasing = F),])
}

For better overview and easier comparison of the top significant pathways, barplot and dotplot are employed. Functions allow to generate these plots. 

# Functioin for making bar and dot plot
pathwayBarPlot <- function(finTab){
  pathways <- rep(c(finTab[,2]),3)
  allPathGen <- finTab[,3]-finTab[,4]
  noSigGen <- finTab[,4]-finTab[,5]
  sigGen <- finTab[,5]
  values <- c(sigGen, noSigGen, allPathGen)
  type <- c(rep("Significant genes", nrow(finTab)), rep("No significant genes", nrow(finTab)), 
            rep("All pathway genes", nrow(finTab)))
  mydata <-data.frame(pathways, values)
  mydata$pathways <-factor(mydata$pathways, 
                           levels = c(finTab[,2]))

  p <-ggplot(mydata, aes(pathways, values))
  p +geom_bar(stat = "identity", aes(fill = type)) + coord_flip() + 
  ggtitle("Dotplot of the pathways") + ylab('Number of genes') + xlab('Pathways') +
  theme(axis.text=element_text(size=12),axis.title=element_text(size=12,face="bold"),title=element_text(size=12,face="bold"))
}

pathwayDotPlot <- function(finTab){
  df <- fortify(finTab, showCategory = 10, split=split)
  df$GeneRatio <- finTab$OurGen/finTab$PathSize
  idx <- order(df$GeneRatio, decreasing = FALSE)
  df$PathName <- factor(df$PathName, levels=unique(df$PathName[idx]))
  ggplot(df, aes_string(x="GeneRatio", y="PathName",size="NrSigGen", color="Pval")) +
    geom_point() + scale_color_gradient(low="red", high="yellow") + ylab('Pathways')  +
    ggtitle("Dotplot of the pathways")+ xlab('Gene ratio (experimnet vs. pathways)') +
    theme(axis.text=element_text(size=12),axis.title=element_text(size=12,face="bold"),title=element_text(size=12,face="bold"))
}

Pathway analysis - gene network reconstruction

Although we have sparse pathways representation (small number of genes measured in our experiment which represent particular pathways), we proceed with pathway analysis including genes gene counts. We are interested to reconstruct interaction among genes from breast cancer pathways using measurements of the genes from the experiment. Although we have only 32 genes from breast cancer pathways defined by KEGG, we can still investigate how they interact among each other. Especially we aim to identify genes which are regulators, due to the fact this genes are consider to be potential cancer driving genes. Methodology employed here is graphical modeling, which use ridge penalized ML estimator for precision matrix in order to identify all the interactions among the genes. Graphical modeling is based on p-dimensional Gaussian multivariate distribution, which we do not have in our data. In order to assume this distribution we previously Gaussianize our data transferring from negative binomial to normal.

# Reconstruction of the interaction among the genes from Breast cancer

# Gausianize data
X.npn = huge.npn(t(brestDat))

## Conducting the nonparanormal (npn) transformation via shrunkun ECDF....done.

Cx <- covML(X.npn)

# Fit ridge penalized maximum likelihood model
ridgeEst <- ridgeP(Cx, lambda = 1, type = "Alt")

# Sparsify the precision matrix, identifying only significant gene interactions.

From the both enrichment and network reconstruction analysis, we can conclude that it is required to enlarge our selection of genes from the experiment. This will allow us to have better overview of the over-represented pathways, as well as for network reconstruction to identify genes with highest number of interactions. These genes may be considered as potential cancer driving genes and can be further investigated.
