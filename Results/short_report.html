Short report

Estrogen impact on differential expression

The objective of this test was to find differentially expressed genes indeced by the presence of estrogen. To find such genes only control samples were used. Models tested here were: full ~ plate + estrogne vs reduced ~ plate. This and all subsequent test used nbinomLRT function from DESeq2 package. Note that in this setting we were controling for 'plate' variable which is justified by the fact that each plate contains both estrogen starved and estrogen stimulated control samples. In tests when one of the variables of interest is chemical plate variable shoudn't be used for controling as it contains information about chemical (on each plate there are exactly two chemicals not all of them). Thus, in such cases we will be controling just for bilogical replicate.

dds_control = dds[, colData(dds)$chemical == 'control']
dds_control$chemical <- droplevels(dds_control$chemical)

dds_control = nbinomLRT(dds_control, 
                        full = ~ plate + estrogen,
                        reduced = ~ plate)

res_positive_control <- results(dds_control)
diff_expressed_genes_estrogen_driven = rownames(res_positive_control)[which(res_positive_control$padj < 0.05)]

print(paste("Number of differentially expressed genes:", toString(sum(res_positive_control$padj < 0.05, na.rm = T))))

## [1] "Number of differentially expressed genes: 302"

It can be seen that estrogen itself led more than a half of genes (302 out of 557 tested) to be differentially expressed. This set of differentially expressed genes will be used for determing which chemical has expressed 'strogen like' charecteristics.

Individual chemical chemical impact on differential expression (in the ABSENCE of estrogen)

Here we want to see the pure impact (without presence of estrogen) of each chemical on differential expression. For this setting only estrogen starved samples were kept. Each chemical is separately compared to control samples (note that chemicals samples were treated as if they were present at the same concentration (i.e. no controling for dosage). This approach has been chosen in order to increase statistical significance of obtained results. Results are shown in the table bellow. Each column in the table refers to results obtained by using particular combination of doses. Numbers in brackets refer to intersection of differentially expressed genes induced by estrogen and corresponding chemical.

Obviousy, BQ, PFHxA PFNA and PFOA have no or negligible effect on differential expression at any dosage. On the other hand chemicals BBP, BPA, Gen and Tam expressed strong differentail expression signal. Moreover, when looking at the third column where strongest signal is expected to be observed, Tam chemical has much high percentage of differentially expressed genes that were not affeced by estrogen (36 %), while this percentage for other 3 chemicals ranges from 4-8 %.

Testing if there is any any impact induced by chemical at a dosage level of 0.001

As it may be seen in the first table, excluding lowest dose 0.001 have led to significantly higher signal in all 4 influential chemicals. That led us to make a hypothesis that dosage level of 0.001 has no impact on DE. To test this hypothesis we kept only estrogen starved samples, control samples and chemical treated samples at 0.001 dosoage level (code shown bellow).

dds_custom = dds[, colData(dds)$estrogen == 'starved']
dds_custom = dds_custom[, colData(dds_custom)$concentration %in% c(0.001, 0)]
dds_custom$estrogen = droplevels(dds_custom$estrogen)

dds_custom <- nbinomLRT(dds_custom, 
                        full = ~ biological_replicate + chemical,
                        reduced = ~ biological_replicate)

res_control_vs_chemichals_at_0.001 <- results(dds_custom)

diff_expressed_genes = rownames(res_control_vs_chemichals_at_0.001)[which(res_control_vs_chemichals_at_0.001$padj <= 0.05)]

Disclaimer

Though, we made some conclusions about dosage (the previous one) dosage effect can be further invastigated. There are foew questions that remained unaswered, for example: - Which chemical has the highest impact at the lowest doasege? - The saturation level of each chemical (in the presence and absence of estrogen). However, in this experiment design it is not posiible to answer these as there were no enough samples to provide decent statistical significance.
